## 功能介绍

 * 编码转换
   * Ascii 编码
   * Web 常用编码
   * Hex 编码
   * Unicode 编码
   * Base64 编码
   * Url 编码
   * Html Entity 编码
   * Escape 编码
   * Quoted-printable 编码
   * Morse Code（莫尔斯电码）
   * Tap Code（敲击码）
 * 古典密码
   * Rail-fence（栅栏密码）
   * Atbash（埃特巴什码）
   * Caesar（凯撒密码）
   * ROT13
   * Simple Substitution（简单替换密码）
   * Hill（希尔密码）
   * Polybius Square （波利比奥斯方阵密码）
   * Playfair（普莱菲尔密码）
   * Vigenère（维吉尼亚密码）
   * Autokey（自动密钥密码）
   * Beaufort（博福特密码）
   * Running Key（滚动密钥密码）
   * Porta（Porta 密码）
   * Affine（仿射密码）
   * Baconian（培根密码）
   * ADFGX（ADFGX 密码）
   * ADFGVX（ADFGVX 密码）
   * Bifid（双密码）
   * Four-Square（四方密码）
   * Straddle Checkerboard （跨棋盘密码）
 * 密码学
   * 常用 Hash
   * MD5（MD2、MD4、RIPEMD)
   * SHA（SHA1、SHA256、SHA512、SHA224、SHA384）
   * AES
   * DES
   * 3DES（TripleDES）
   * RC4
   * Rabbit
   * Blowfish
 * 文本
   * RegExp Tester（正则表达式）
 * 图像
   * Base64 图像编码
 * utility
   * Mime Type
